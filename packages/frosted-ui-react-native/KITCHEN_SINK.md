# Component Kitchen Sink

A comprehensive showcase of all React Native Reusables components available in this library.

## Access

Navigate to `/kitchen-sink` in your app to view all components.

## Components Showcased

### Typography
- Text variants (h1, h2, h3, h4, p, lead, large, small, muted, code, blockquote)

### Buttons
- Button variants (default, secondary, destructive, outline, ghost, link)
- Button sizes (sm, default, lg, icon)
- Buttons with icons
- Disabled state

### Badges
- Badge variants (default, secondary, destructive, outline)

### Cards
- Card with header, content, and footer
- CardTitle and CardDescription

### Alerts
- Default and destructive variants
- With icons

### Form Inputs
- Input (with label)
- Textarea
- Disabled state

### Form Controls
- Checkbox (with label)
- Radio Group
- Switch
- Toggle
- Toggle Group

### Layout
- Separator
- Aspect Ratio

### Feedback
- Progress bars (0%, 25%, 50%, 75%, 100%)
- Skeleton loaders
- Alert messages

### Data Display
- Avatar (with image, fallback, and icon)
- Accordion (collapsible sections)
- Tabs (with tab content)

### Overlays
- Dialog (modal dialog)
- Alert Dialog (confirmation dialog)
- Popover (floating content)
- Dropdown Menu (with checkbox and radio items)
- Select (dropdown select)
- Context Menu (right-click/long-press menu)
- Menubar (menu bar navigation)
- Hover Card (hover-triggered content)
- Tooltip (hover-triggered tooltip)

### Interactive
- Collapsible (expandable content)

## Usage

Each component section demonstrates:
- All available variants
- Different sizes (where applicable)
- Different states (active, disabled, etc.)
- Real-world usage examples

## Development

This kitchen sink page is useful for:
- **Component Library Development**: See all components at a glance
- **Visual Regression Testing**: Catch UI changes across all components
- **Documentation**: Reference implementation examples
- **Accessibility Testing**: Test all components for accessibility
- **Theme Testing**: Quickly test light/dark mode across all components
